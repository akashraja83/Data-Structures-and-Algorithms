## 🚀 How to Use

```bash
# 1. Clone the repository
git clone https://github.com/akashraja83/Data-Structure-and-Algorithm.git

# 2. Navigate to a topic
cd "Data-Structure-and-Algorithm/Linear Data Structures/Linked List/Singly Linked List"

# 3. Compile the Java file
javac SinglyLinkedList.java

# 4. Run it
java SinglyLinkedList
```

> **Requirements:** 
> - Java 8 or higher
> - Any IDE (IntelliJ IDEA, VS Code, Eclipse, or just terminal)
> - Git (for cloning)
